KG is an open source 3d game engine. The engine is in development. This website is under construction.

Important!

It is very important to know this project is in a very early stage of development.

What is KG engine?

KG is an open source 3d game engine intended to be powerful and flexible and still have an easy to use API. The Engine is game-object component oriented, flexible and designed to be easy and intuitive for developers. It is plugin based, manager based and will have an intuitive editor.

Is KG a game engine?

Yes. KG is a 3d game engine. It is inspired from OGRE's power and Irrlicht's easy API, CryEngine's level editor and Unreal Editor's resource management.

Is it really free?

Yes. The KG engine is under the MIT License.
